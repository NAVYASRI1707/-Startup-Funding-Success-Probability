# Startup Funding Success Probability

## Problem Statement

Thousands of startups are launched every year, but only a small percentage secure funding and achieve sustainable growth. Investors and startup accelerators require data-driven insights to identify patterns associated with successful funding outcomes. This project analyzes startup funding data to understand investment trends and factors influencing funding decisions.

## Dataset Description

The project uses the Indian Startup Funding Dataset (2020–2025), containing 1100 startup funding records. The dataset includes the following features:

* Startup
* Industry
* SubVertical
* City
* Investors
* InvestmentType
* InvestmentAmount_USD
* Date

The dataset provides information about startup investments across various industries, cities, and funding stages.

## Statistical Methods

The following statistical techniques were applied:

* Data Cleaning
* Missing Value Analysis
* Duplicate Removal
* Descriptive Statistics

  * Mean
  * Median
  * Standard Deviation
  * Quartiles
* Frequency Distribution
* Probability Analysis
* Exploratory Data Analysis (EDA)
* Data Visualization

## Probability Findings

* Seed funding was the most common investment type.
* FoodTech startups received the highest overall funding.
* Consumer Electronics and Retail sectors attracted significant investments.
* Funding activity was concentrated in specific startup hubs and industries.

## Visual Insights

The project includes visualizations such as:

* Top Industries by Funding
* Top Cities by Funding
* Investment Type Distribution
* Industry-wise Funding Comparison

These visualizations help identify investment trends and funding patterns within the startup ecosystem.

## Recommendations

* Focus on high-growth sectors such as FoodTech and Consumer Electronics.
* Encourage data-driven investment decisions using historical funding trends.
* Support startups in emerging industries with strong growth potential.
* Monitor funding activity across cities to identify expanding startup ecosystems.

## Future Scope

* Develop machine learning models to predict startup funding success.

## Conclusion

This project demonstrates how statistics, probability, and data analytics can be used to understand startup funding patterns and generate actionable insights for investors, accelerators, and business decision-makers.
